my note on setting up openCV3.1 with visual studio on win10
1. environment variable: Path.
add -- >
C:\opencv\build\x64\vc14\bin
to path. this is the link to the opencv bin... opencv3.1 only supports 64 bit computer
2. add libraries:
with VS15, create sourced.cpp
open view -> other windows -> property manager -> Debug x64 -> right click -> properties -> VCC directories -> include directories
add the following
C:\Opencv3.1.0\opencv\build\include
C:\Opencv3.1.0\opencv\build\include\opencv
C:\Opencv3.1.0\opencv\build\include\opencv2
in library directories: add
C:\Opencv3.1.0\opencv\build\x64\vc14\lib

same window, click on Linker -> input -> additoinal dependencies, add
opencv_world310d.lib
